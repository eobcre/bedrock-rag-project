# AWS x RAG

### 🚀 Features 🚀

- topK control
- Semantic search

### 🛠 Tech Stack 🛠
#### Frontend
- React (Vite)
- Tailwind

#### Backend / AWS
- Amazon Bedrock
- Amazon Bedrock Knowledge Base
- AWS Lambda
- Amazon API Gateway
- Amazon S3 Vectors
